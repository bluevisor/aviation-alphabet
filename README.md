# Aviation Alphabet

Learn and practice the NATO phonetic alphabet.

## Features

- **Learn**: Full list of the alphabet (A-Alpha, B-Bravo...) with pronunciation.
- **Practice**: Interactive flashcards to test your memory.
- **Minimalist Design**: Inspired by ukulele.cc.

## Tech Stack

- Next.js 14+
- Tailwind CSS
- TypeScript
- Vercel Deployment

## Getting Started

```bash
npm install
npm run dev
```

## Deployment

Deployed on Vercel.
